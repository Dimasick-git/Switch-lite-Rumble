# hardware/

Hardware design files and references for the Switch-lite-Rumble project.

| Folder | What it is |
| :--- | :--- |
| [`MIG-reference/`](./MIG-reference/) | KiCad projects, schematics, boardviews and BOMs for the **MIG Dumper & Flashcart**, imported as prior art (original by [sabogalc](https://github.com/sabogalc/MIG-Flash-PCBs), WTFPL). The Flashcart is the closest existing slot-fit, Lotus3-speaking board to what this project needs. |

For the chip selection and the physical/electrical constraints these designs have
to live within, see [`../CHIPS.md`](../CHIPS.md).
